How to Compile
In the project directory, run:

make clean
make
This will build the executable called pseudo-shell.

How to Run
Interactive Mode:

./pseudo-shell
You will see the prompt >>> . Type commands one at a time.
Type exit to quit the shell.

Example:

>>> pwd
>>> ls
>>> exit

File Mode:
./pseudo-shell -f input.txt
